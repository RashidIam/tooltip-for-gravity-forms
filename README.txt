Add Tooltips next to field labels of Gravity Forms.

== Description ==

Add tooltips next to field labels very easily.

How to use:

*   Add your text to the "Tooltip Text" field while editing gravity forms fields.
*   Save changes and tooltips will appear next to label automatically in the form.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `gravity-forms-tooltip` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
